API ENDPOINTS (Flask)
=====================

1. POST /api/translate
   - Single text translation
   - Auto-language detection
   - Confidence scoring

2. POST /api/batch-translate
   - Multiple texts (array)
   - Batch processing
   - Efficient for many translations

3. POST /api/detect-language
   - Identify if English or Luganda
   - Returns confidence

4. GET /api/history
   - Get recent translations
   - Configurable limit
   - With timestamps

5. GET /api/phrasebook
   - Common phrases by category
   - Learn Luganda
   - JSON format

6. GET /api/status
   - System health check
   - Device info (GPU/CPU)
   - Model information

7. DELETE /api/clear-history
   - Clear translation database
   - Reset history
